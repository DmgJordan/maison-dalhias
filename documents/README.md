# Dossier Documents

Ce dossier contient les templates et documents de référence pour la location "Maison Dalhias 19".

## Contenu

### Templates de documents
- `contrat-exemple.pdf` - Template de contrat de location
- `facture-exemple.pdf` - Template de facture

## Usage

Ces documents servent de modèles pour :
1. **Contrat de location** : À personnaliser pour chaque réservation avec les informations client et dates
2. **Facture** : À adapter selon les tarifs et services choisis

## Notes importantes

- Ces documents sont des exemples à adapter selon vos besoins spécifiques
- Veillez à respecter la réglementation française en matière de location de vacances
- Consultez un professionnel juridique pour valider vos contrats

## Gestion des documents

Pour générer automatiquement ces documents, vous pouvez :
1. Utiliser la fonction PDF de l'interface admin
2. Intégrer un système de templates personnalisables
3. Connecter avec un service de génération de documents
